# AdvancedCalculator

**AdvancedCalculator** is a simple C++ console-based calculator designed for beginners in Computer Science.  
It allows users to perform basic arithmetic operations with a user-friendly, menu-driven interface.

## Features
- Addition (+), Subtraction (-), Multiplication (*), Division (/)
- Input validation to prevent invalid entries
- Handles division by zero gracefully
- Looping menu for multiple calculations without restarting

## How to Run
1. Download or clone this repository.
2. Open terminal in the project folder.
3. Compile the program:

```bash
g++ calculator.cpp -o calculator

4. Run the program:
./calculator   # On Windows: calculator.exe

5. Follow the menu to perform calculations and choose 'Exit' when done.
Purpose
This project helps first-semester Computer Science students practice C++ basics such as functions, loops, and console input/output.
It also demonstrates good coding practices like modular design and input validation.
